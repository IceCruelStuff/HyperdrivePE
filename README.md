#### Pocketmine API Support
- 3.0.0-ALPHA9
- 3.0.0-ALPHA10

## Commands
- /hyper create [world] - Creates a hyperdrive game in the specified world.
- /hyper quit - Allows the player to quit the game while spectating or while waiting for game to start.
